## Aniquote

It returns the famous dialogs of anime characters after entering their names.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install aniquote
```

## Usage

```python
from aniquote import quotes

quotes.quote()
```
